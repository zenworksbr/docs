*************************************************************
Correção para páginas WEB do servidor exibidas incorretamente
*************************************************************

 Para corrigir o problema das páginas WEB não carregarem de forma correta dentro do servidor, você precisa entrar na beta **x86-64 - Chromium + 64-bit binaries**, e fazer isso é extremamente fácil e rápido.

 **Guia prático em etapas:**

 - **Feche o jogo;**

 - **Na biblioteca steam, clique com o botão direito sobre o Garry's Mod;**

 - **Selecione Propriedades;**

 - **Selecione Betas; e**

 - **Em betas, selecione x86-64 - Chromium + 64-bit binaries**.

 Seu jogo irá instalar uma pequena atualização, após isso, ao abrir o jogo, será mostrado 3 opções de inicialização.

 - **Jogar Garry's Mod;**

 - **Launch Garry's Mod (32-bit); e**

 - **Launch Garry's Mod (64-bit)**.

 Selecione **Launch Garry's Mod (64-bit)** e clique em **Usar sempre esta opção**.
